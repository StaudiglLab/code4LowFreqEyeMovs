---------------
Helper functions for analysis
---------------

ang2px.m: convert the visual angle to number of pixel on the screen based on the distance to the screen

BilateralOverlap.m: return the overlaping or combining timepoints of each saccade detected in both eyes

Gaze_balance_trial.m: bootstrap to balance the trial numbers between two conditions for gaze density data

MarkTFRNans_ft.m: mark the artifacts as NaNs for intracranial EEG data

TFR_balance_trial.m: bootstrap to balance the trial numbers between two conditions for time frequency data

visAngPerPixel.m: compute the visual angle per pixel on the screen based on the distance to the screen
